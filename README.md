## Installation

### Quick Install

```sh
curl -fsSL https://raw.githubusercontent.com/grexlabs/real_grexcode/refs/heads/main/install | bash
```

### Package Managers

```sh
npm i -g grexcode-ai@latest
scoop install grexcode
choco install grexcode
brew install grexlabs/tap/grexcode
sudo pacman -S grexcode
paru -S grexcode-bin
mise use -g grexcode
```

### Binary Downloads

| Platform | Binary |
|----------|--------|
| Linux x64 | `grexcode-linux-x64` |
| Linux ARM64 | `grexcode-linux-arm64` |
| macOS Apple Silicon | `grexcode-darwin-arm64` |
| macOS Intel | `grexcode-darwin-x64` |
| Windows x64 | `grexcode-windows-x64.exe` |

Download from [GitHub Releases](https://github.com/grexlabs/real_grexcode/releases).

## Agents

Grexcode includes two built-in agents:
- **Build** -- Default agent for coding tasks
- **Plan** -- For planning and analysis

Switch between them with the `Tab` key.
